mmjiostemplates

  • 通用App iOS模板，不含故事板或XIB，具有各种尺寸的Launchscreen。对于Xcode 6.x
  • 这是有据可查的，所以它可能是制作您自己的Xcode 6.x iOS项目模板的起点。

＃＃它是什么

  • 基本的通用iOS应用模板。
  • 它附带AppDelegate，Prefix.pch，Info.plist和Localizable.strings。
  • 还包括具有AppIcon和LaunchImage的所有尺寸的Images.xcassets。
  • 包含AppIcon和LaunchImage的草图文件。

##模板包含两个模板：### MMJ Base通用iOS应用程序

  • 它只有基本的文件：AppDelegate，Prefix.pch，Info.plist和Localizable.strings。
  • 它包括AppDelegate中的导航控制器。

### MMJ令牌通用iOS应用程序

  • 它拥有“MMJ基础通用iOS应用程序”模板的一切。
  • 还包括数据源模式的基本模型类。
  • BaseViewController。
  • 基本视角。
  • BaseSubview是用于嵌入ViewController的主视图类。它具有动态确定其大小的方法。
  • BaseTableViewCell，BaseCollectionViewCell和BaseTableViewHeaderFooterView类也有动态确定其大小的方法
  • 所有需要自动布局的课程都包含了砖石，但已经评论过。
  • Podfile包含在我最喜欢的Cocoapods中。你可以改变你的喜好。

＃＃如何安装

  • 将整个目录复制到：/ Users / {您的用户名} / Library / Developer / Xcode / Templates / Project Templates

##创建新项目

  • 从“新建项目”对话框中选择“MMJ Base Universal iOS应用程序”或“MMJ Token通用iOS应用程序”。
  • 输入您的“产品名称”和“组织名称”。
  • 创建项目后，更改“边界标识符”。
  • 从每个文件顶部的所有注释中替换我的姓名和公司。如TODO项目所述，当innid所有包含的文件时，它将被修复。

单位测试

  • 我使用Kiwi，所以我没有包含测试目标的主要.m文件。
  • 您可以通过创建一个新文件并选择“测试用例类”来实现。

## LaunchImages和AppIcon

  • 当您使用任何包含的模板创建应用程序时，它附带了所有尺寸的AppIcon和LaunchImage，包含在Images.xcassets中。他们都是紫色的:)
  • 这样做的目的是使应用程序占据任何iOS设备的整个屏幕，并且您可能会看到它，因为在启动应用程序时出现紫色屏幕。
  • 在顶部目录中，包含两个Skecth文件，以便您可以自定义屏幕和图标。
  • 然后只需导出屏幕/图标，并将它们拖放到Images.xcassets中的适当位置。
